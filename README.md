# Zig Todo `--WIP--`

A simple todo list written in Zig. 
> This is a learning project for me to learn Zig.

![image](assets/Screenshot%202023-02-15%20234835.jpg)

## Building

- requires Zig <= `0.11.0-dev.1580+a5b34a61a`
- requires ZigMod

```bash
zigmod fetch
zig build
```

## Running

```bash
zig build run
```
